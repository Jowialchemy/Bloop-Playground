# Bloop Playground 🎈

A fun, colorful learning playground for toddlers and preschoolers (ages 1-5). Pop bubbles, match colors, and sort shapes!

---

## 🎮 Games

| Game | Age | Skill |
|------|-----|-------|
| 🫧 Bubble Pop | 1-3 | Motor Skills |
| 🎨 Color Match | 2-5 | Color Recognition |
| 🔷 Shape Sorter | 3-5 | Pattern Matching |
