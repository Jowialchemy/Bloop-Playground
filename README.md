# Bloop Playground 🎈

Bloop Playground is a colorful educational mini-game collection designed for toddlers and preschoolers ages 1–5.

Kids can:
- 🫧 Pop floating bubbles
- 🎨 Match colorful objects
- 🔷 Learn shapes through play

Built with pure HTML, CSS, and JavaScript for fast performance on phones, tablets, and desktop browsers.

---

## 🎮 Games

| Game | Age | Skill |
|------|------|------|
| 🫧 Bubble Pop | 1–3 | Motor Skills |
| 🎨 Color Match | 2–5 | Color Recognition |
| 🔷 Shape Sorter | 3–5 | Shape & Pattern Recognition |
